Clase 1: Introducción a PHP

Objetivos: comprender qué es PHP, diferencias con Java, conexión a bases de datos.

Empezamos

  • Preparar XAMPP con puerto 3307

¿Qué es PHP?

  • Lenguaje del lado del servidor
  • Amplio uso en desarrollo web (WordPress, Laravel)
  • Sintaxis similar a C / Java
  • Archivos terminan en .php
  • El código PHP se encierra entre <?php ?>

Primer script

<?php
echo "Hola mundo";
?>

Se ejecuta en el servidor y devuelve HTML.

Variables y tipos

$nombre = "Ana";
$edad = 20;
$activo = true;

Estructuras básicas

if ($edad >= 18) {
    echo "Mayor de edad";
} else {
    echo "Menor de edad";
}
for ($i = 0; $i < 5; $i++) {
    echo $i;
}

Funciones

function saludar($nombre) {
    return "Hola, $nombre";
}
echo saludar("Luis");

Superglobales

  • $_GET: datos de la URL
  • $_POST: datos enviados por formulario
  • $_SERVER: info del servidor

Conexión con MySQL (MySQLi)

$conn = new mysqli("localhost", "root", "", "basedatos", 3307);
if ($conn->connect_error) {
    die("Error: " . $conn->connect_error);
}

Conexión con MySQL (PDO)

try {
    $pdo = new PDO("mysql:host=localhost;port=3307;dbname=basedatos", "root", "");
} catch (PDOException $e) {
    echo "Error: " . $e->getMessage();
}
